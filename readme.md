# Haversack

A player-side D&D reference and helper program.

# heck

## Features:

- Inventory tracking
  - Rope
  - SRD + Homebrew items
  - Components and Kits
- Coin bag
  - Money conversion
- Spell resources
  - Spell book reference (importable, homebrewable)
  - Spell slots
  - Attack roll calculators
- Homebrew items
- Stats and Roll Calculators
  - Proficiencies
- Feature reference and tracking
  - Feature charges
- Journaling
  - Notes on items
  - Session notes
